# Face generator

Create different expressions to your model.

## Download on civitai

<a href="https://civitai.com/models/1182357">Face Generator</a>

## How to use.

* Upload the image.
* Define base model.
* Define mask in face of your model.
* Set output folder name.
* Generate faces.
